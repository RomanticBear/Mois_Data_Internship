   - 특정 정책 영역에서는 특정 정당이 압도적으로 높은 관심도 보임
   - 정당 간 정책 관심도 격차가 존재

   - 각 정당의 고유한 정책 아이덴티티 확인 가능
   - 정당별 정책 우선순위의 차이점 명확히 드러남

1. **키워드 기반 분석의 한계**
   - 단순 키워드 매칭으로는 문맥적 의미 파악 어려움

2. **정당 분류의 한계**

   - 정당 소속이 불분명한 발언자들의 발언 제외

3. **시간적 변화 미고려**
