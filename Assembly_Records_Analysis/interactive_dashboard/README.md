# 제415회 국회 회의록 인터랙티브 대시보드

best_books_21c 스타일의 인터랙티브 웹 시각화 대시보드입니다.

## 🎯 기능

1. **핵심 이슈 중요도 차트**
   - 이슈별 중요도 시각화
   - 호버 시 상세 정보 표시
   - 정당별 언급 현황 표시

2. **정당별 주요 관심사**
   - 정당별 관심사 수 비교
   - 정당 필터링 기능
   - 호버 시 주요 발언 요약 표시

3. **질의-응답 효과성 분석**
   - 응답 품질 분포 파이 차트
   - 질문 유형 분포 막대 차트
   - 인터랙티브 호버 효과

4. **안건별 정당 입장 히트맵**
   - 안건별 정당 입장 시각화
   - 색상으로 입장 구분 (지지/중립/반대)
   - 호버 시 상세 정보 표시

5. **정당 간 관계 네트워크**
   - 정당 간 대립/토론/협력 관계 시각화
   - 드래그 앤 드롭으로 네트워크 조작 가능
   - 관계 유형별 색상 구분

## 🚀 사용 방법

### 1. 로컬 서버 실행

Python을 사용하는 경우:
```bash
cd Assembly_Records_Analysis/interactive_dashboard
python -m http.server 8000
```

또는 Node.js를 사용하는 경우:
```bash
cd Assembly_Records_Analysis/interactive_dashboard
npx http-server -p 8000
```

### 2. 브라우저에서 접속

```
http://localhost:8000
```

## 🔄 업데이트 방법

다른 회차의 데이터를 시각화하려면:

1. `dashboard.js`의 `loadData()` 함수에서 JSON 파일 경로 수정
2. 또는 동적으로 회차를 선택할 수 있도록 UI 추가

## 📝 참고사항

- 데이터는 `analysis_results/제415회_openai_analysis.json`에서 로드됩니다
- CORS 정책으로 인해 로컬 파일을 직접 열 수 없습니다 (file:// 프로토콜)
- 반드시 웹 서버를 통해 접속해야 합니다
